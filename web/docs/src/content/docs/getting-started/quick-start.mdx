# Quick Start

Get your first Strata program running right now.

## 1. Install Strata

Follow the [installation guide](./installation) to set up Strata on your system.

Verify it's working:
```bash
strata --version
```

Stuck? Check the [installation troubleshooting section](./installation#troubleshooting).

## 2. Hello, World!

<Steps>
  1. Create a file called `hello.str`:
     ```strata
     import io from str

     io.print("Hello, Strata!")
     ```

  2. Run it:
     ```bash
     strata hello.str

  3. You should see:
     ```
     Hello, Strata!

## 3. The Basics

### Variables and Types

Strata is statically typed. Every variable has a type:

<Tabs syncKey="vartype">
  <TabItem label="Immutable">
    ```strata
    let x: int = 42
    let name: string = "Alice"
    let value: float = 3.14
    let flag: bool = true
    ```

    Use `let` for variables that never change.
  </TabItem>

  <TabItem label="Mutable">
    ```strata
    var count: int = 0
    var count: int = count + 1
    var count: int = count + 1
    ```

    Use `var` when you need to reassign. Redeclare with `var` each time!
  </TabItem>
</Tabs>

<Aside type="note">
In Strata, you must redeclare with `var` each time you reassign. This is intentional for clarity!
</Aside>

### Functions

Define functions with `func`:

```strata
func greet(name: string) => string {
  return "Hello, " + name + "!"
}

import io from str
io.print(greet("World"))  // Hello, World!
```

The `=> type` is the return type.

## 4. A Complete Example

Let's write a more substantial program:

```strata
import io from str

// Define a function
func factorial(n: int) => int {
  var result: int = 1
  for (var i: int = 1; i <= n; var i: int = i + 1) {
    var result: int = result * i
  }
  return result
}

// Use it
let num: int = 5
let fact: int = factorial(num)
io.print(fact)  // 120
```

<Aside type="note">
Notice how:
- Types are explicit: `func name(param: type) => returnType`
- Variables use `let` or `var`
- `var` is redeclared each time you reassign
- Functions have clear signatures
</Aside>
