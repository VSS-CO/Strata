# Variables and Types

Strata is a statically typed language. Every value has a type that's checked at compile time.

## Declaring Variables

### Immutable Variables (`let`)

Use `let` for values that never change:

```strata
let x: int = 42
let name: string = "Alice"
let value: float = 3.14
```

Once assigned, you cannot reassign a `let` variable:

```strata
let x: int = 10
var x: int = 20  // Error: x already defined
```

### Mutable Variables (`var`)

Use `var` when you need to reassign:

```strata
var count: int = 0
var count: int = count + 1  // OK
var count: int = count + 1  // OK again
```

<Aside type="danger" title="Important: Redeclaration">
In Strata, you must redeclare with `var` each time you reassign. This is intentional for clarity:

```strata
var x: int = 5
var x: int = x + 1  // ✅ Correct - redeclare
x = x + 1           // ❌ Error - cannot use assignment without var
```
</Aside>

### Best Practice

<Card title="Use `let` by default">
Only use `var` when you need to reassign:

```strata
// Good
let items: int = 10
let name: string = "Product"

// Use var only when necessary
var total: int = 0
var total: int = total + items
```
</Card>

## Primitive Types

<Tabs syncKey="types">
  <TabItem label="int">
    32-bit signed integer:

    ```strata
    let count: int = 0
    ```

    ```strata
    let sum: int = 5 + 3         // 8

    let remainder: int = 10 % 3  // 1
    ```

  <TabItem label="float">
    64-bit floating-point number:
    ```strata
    let pi: float = 3.14159
    let temperature: float = 98.6
    let price: float = 19.99
    ```

    **Operations:**
    ```strata
    let result: float = 10.5 + 2.5     // 13.0
    let average: float = (5.0 + 10.0) / 2.0  // 7.5
    ```

  <TabItem label="string">
    Text values:

    **Concatenation:**
    ```strata

    ```
  </TabItem>

  <TabItem label="bool">
    Boolean values - `true` or `false`:

    ```strata

    if (!a) { ... }      // NOT → false
    ```

## Arrays

Arrays store multiple values of the same type:

```strata
let numbers: any = [1, 2, 3, 4, 5]
let colors: any = ["red", "green", "blue"]
let empty: any = []
```

**Access elements by index (0-based):**
```strata
let first: any = numbers[0]   // 1
let second: any = numbers[1]  // 2
```

**Get array length:**
```strata
import io from str
let length: any = io.len(numbers)  // 5
```

## Type Safety in Functions

Functions must declare their types:

```strata
func add(a: int, b: int) => int {
  return a + b
}

func greet(name: string) => string {
  return "Hello, " + name
}

func isEmpty(s: string) => bool {
  return s == ""
}
```

<Aside type="note" title="Type Checking">
Strata checks types at compile time. This prevents many bugs:

```strata
let result: string = add(5, 3)  // Error: int cannot be assigned to string
```
</Aside>
