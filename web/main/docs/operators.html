Operators

Operators are symbols that perform operations on values. Strata supports arithmetic, comparison, logical, and bitwise operators.

Arithmetic Operators

Operator Description Example Result
+ Addition 10 + 5 15
- Subtraction 10 - 5 5
* Multiplication 10 * 5 50
/ Division 10 / 5 2
% Modulo (remainder) 10 % 3 1

Comparison Operators

Comparison operators return a boolean (true or false):

Operator Description Example
== Equal to x == 5
!= Not equal to x != 5
< Less than x < 5
> Greater than x > 5
<= Less than or equal x <= 5
>= Greater than or equal x >= 5

Logical Operators

let x: bool = true let y: bool = false x && y // false (AND: both must be true) x || y // true (OR: at least one true) !x // false (NOT: negate boolean)

Operator Precedence

Operators are evaluated in this order (highest to lowest):

Precedence Operators
1 (Highest) !, -, + (unary), ~ (bitwise NOT)
2 *, /, % (multiplicative)
3 +, - (additive)
4 <, >, <=, >= (relational)
5 ==, != (equality)
6 && (logical AND)
7 (Lowest) || (logical OR)

String Concatenation

let greeting: string = "Hello" + " " + "World" io.print(greeting) // "Hello World"
