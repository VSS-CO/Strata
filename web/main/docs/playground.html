Strata Playground

The Strata Playground is a web-based editor where you can write and run Strata code directly in your browser - no installation required.

Open Playground →

Features

  • Live Editing: Write code and see results instantly
  • Built-in Examples: 10+ curated examples to learn from
  • Syntax Highlighting: Code with proper highlighting
  • Output Panel: See execution results in real-time
  • Example Browser: Search and load examples easily
  • Run Button: Execute code with Ctrl+Enter or click

Getting Started

  1. Click the Open Playground button above
  2. Select an example from the left sidebar
  3. Edit the code in the editor
  4. Press Ctrl+Enter or click ▶ Run
  5. See your output on the right

Tips & Tricks

  • Keyboard Shortcut: Ctrl+Enter runs your code
  • Search Examples: Use the search box to find examples
  • Switch Tabs: Click Editor/Output to switch views
  • Copy Code: Select all text and copy for use elsewhere

Examples in Playground

The playground includes these built-in examples:

  • Hello World - Print to output
  • Basic Types - Primitive types and variables
  • Functions - Define and call functions
  • Control Flow - If/else and loops
  • Arrays - Collections and iteration
  • Math Module - Mathematical operations
  • String Operations - Text manipulation
  • Type Safety - Compile-time type checking
  • Immutability - let, const, var differences
  • Nested Control - Complex control structures

Common Tasks

Print Multiple Lines

Call io.print() multiple times:

import io from std::io
io.print("Line 1")
io.print("Line 2")
io.print("Line 3")

Use Math Functions

Import the math module:

import io from std::io
import math from std::math
io.print(math.sqrt(16.0))

Create Arrays

Use square bracket syntax:

let numbers: [int] = [1, 2, 3, 4, 5]
io.print(len(numbers))
