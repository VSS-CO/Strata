Installation

Prerequisites

  • Node.js 14+ (get it from nodejs.org)
  • npm (comes with Node.js)

Install Strata

npm install npx tsc # Compile TypeScript to JavaScript

Your First Program

1. Create a file

Create a file named hello.str:

import io from std::io io.print("Hello, Strata!")

2. Run it

node dist/main.js hello.str

3. See the output

Your program will print to the console:

Hello, Strata!
