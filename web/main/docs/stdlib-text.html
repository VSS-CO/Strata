std::text - String Operations

The text module provides string manipulation functions.

Functions

Function Parameters Returns Description
len(s) string int Get string length
upper(s) string string Convert to uppercase
lower(s) string string Convert to lowercase

Examples

String Length

import io from std::io import text from std::text let message: string = "Strata" let length: int = text.len(message) // 6 io.print(length)

Case Conversion

import io from std::io import text from std::text let message: string = "Hello, World!" io.print(text.upper(message)) // HELLO, WORLD! io.print(text.lower(message)) // hello, world!

String Analysis

import io from std::io import text from std::text func analyzeString(s: string) => void { io.print("String: ") io.print(s) io.print("Length: ") io.print(text.len(s)) io.print("Uppercase: ") io.print(text.upper(s)) } analyzeString("strata")
