std::io - Input/Output

The IO module provides basic input and output operations.

Functions

Function Parameters Returns Description
print(value) any void Print value to stdout with newline

Examples

Print Values

import io from std::io io.print(42) io.print("Hello, World!") io.print(3.14) io.print(true)

Print from Functions

import io from std::io func greet(name: string) => void { io.print("Hello, " + name) } greet("Alice")

Debug Output

import io from std::io let x: int = 10 io.print("x = ") io.print(x)
