Docs / Types

The Type System

Strata is a statically typed language. All variables and function parameters must have explicit type annotations. Type checking happens at compile time before your code runs, catching errors early.

Primitive Types

Strata provides five primitive types for basic data:

Type Description Example Default
int Integer numbers 42, -17, 0 0
float Floating-point numbers 3.14, -2.5, 1.0 0.0
bool Boolean true/false true, false false
char Single character 'A', 'x', '7' '\0'
string Text/characters "hello", "world" ""

Using Type Annotations

In Strata, you must explicitly declare types. The syntax is name: Type:

let age: int = 25 let price: float = 19.99 let active: bool = true let grade: char = 'A' let name: string = "Alice"
Explicit Types Catch Errors: Without explicit types, it's easy to mix incompatible values. Strata requires you to be precise about what types you expect.

Type Compatibility

Strata is strict about type compatibility. You cannot assign a value of one type to a variable of another type:

let x: int = 42 let y: string = x // ERROR: type mismatch

Integer to Float Conversion

One special case: integers can be implicitly converted to floats when needed:

let a: int = 42 let b: float = a // OK: int implicitly converts to float

The "any" Type

For cases where you need flexibility, use the any type:

let value: any = 42 let value: any = "hello" // OK: any accepts anything
Use "any" Sparingly: The any type bypasses type safety. Use it only when absolutely necessary for interoperability or gradual typing.

Arrays

Arrays are ordered collections of a single type. The syntax is [Type]:

let numbers: [int] = [1, 2, 3, 4, 5] let names: [string] = ["Alice", "Bob", "Charlie"]

Accessing Array Elements

Use square bracket notation with zero-based indexing:

let first: int = numbers[0] // 1 let last: int = numbers[4] // 5

Array Length

Get array length with the len() function:

let count: int = len(numbers) // 5

Type Checking in Practice

Strata checks types before executing your code. This catches many common errors:

// Type mismatch detected at compile time let x: int = "42" // ERROR: cannot assign string to int // Array type mismatch let nums: [int] = [1, "2", 3] // ERROR: mixed types in int array // Function return type mismatch func getValue() => int { return "not an int" // ERROR: string is not int }

Summary

  • Strata is statically typed - types checked before execution
  • Explicit annotations required - no type inference
  • Five primitives: int, float, bool, char, string
  • Type strict - no implicit conversions (except int→float)
  • Arrays use [Type] syntax and are type-safe
  • any type available for flexibility when needed
