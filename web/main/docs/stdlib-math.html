std::math - Mathematical Functions

The math module provides mathematical functions for calculations.

Functions

Function Parameters Returns Description
sqrt(x) float float Square root
pow(x, y) float, float float Power (x to the power of y)
abs(x) int/float int/float Absolute value
floor(x) float float Round down to nearest integer
ceil(x) float float Round up to nearest integer
sin(x) float float Sine (radians)
cos(x) float float Cosine (radians)

Examples

Square Root

import io from std::io import math from std::math let result: float = math.sqrt(16.0) // 4.0 io.print(result)

Power and Absolute Value

import io from std::io import math from std::math let p: float = math.pow(2.0, 8.0) // 256.0 let a: int = math.abs(-42) // 42 io.print(p) io.print(a)

Rounding Functions

import io from std::io import math from std::math let x: float = 3.7 io.print(math.floor(x)) // 3.0 io.print(math.ceil(x)) // 4.0
