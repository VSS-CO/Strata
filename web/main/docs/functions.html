Functions

Functions are reusable blocks of code that perform a specific task. In Strata, functions require explicit parameter and return types.

Function Syntax

The basic syntax is:

func name(param1: Type1, param2: Type2) => ReturnType { // function body return result }

Simple Functions

No Parameters, No Return

func greet() => void { io.print("Hello, World!") } greet()

With Parameters

func add(a: int, b: int) => int { return a + b } let result: int = add(10, 20) io.print(result) // 30

Multiple Parameters

func describe(name: string, age: int) => string { return name + " is " + age } io.print(describe("Alice", 30))

Return Values

Functions must specify their return type. Use void for functions that don't return anything:

func double(x: int) => int { return x * 2 } func printDouble(x: int) => void { io.print(double(x)) }

Type Safety in Functions

Function parameters and return types are checked at compile time:

func getAge() => int { return 30 } let age: int = getAge() // OK // let name: string = getAge() // ERROR: int is not string

Recursion

Functions can call themselves recursively:

func factorial(n: int) => int { if (n <= 1) { return 1 } return n * factorial(n - 1) } io.print(factorial(5)) // 120

Function Composition

Functions can call other functions:

func square(x: int) => int { return x * x } func sumSquares(a: int, b: int) => int { return square(a) + square(b) } io.print(sumSquares(3, 4)) // 25
Type Safety Benefits: Explicit parameter and return types mean the compiler catches misuse before your code runs. You'll never call a function with wrong types.
