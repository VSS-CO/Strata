Modules & Imports

Strata has an explicit module system. All code is organized into modules, and modules are imported with explicit syntax.

Standard Library Imports

The standard library is available under the std:: namespace:

import io from std::io import math from std::math import text from std::text import util from std::util

Using Imported Modules

Access functions and constants from an imported module using dot notation:

import io from std::io import math from std::math io.print("Hello") let result: float = math.sqrt(16.0)

Standard Library Modules

std::io

Input/output operations:

  • io.print(value) - Print value to output

std::math

Mathematical functions:

  • math.sqrt(x) - Square root
  • math.pow(x, y) - Power function
  • math.abs(x) - Absolute value
  • math.floor(x) - Floor function
  • math.ceil(x) - Ceiling function
  • math.sin(x), math.cos(x) - Trigonometry

std::text

String operations:

  • text.len(s) - String length
  • text.upper(s) - Convert to uppercase
  • text.lower(s) - Convert to lowercase
Module Namespace: All imports create a namespace. You cannot shadow standard library imports with user code.
