Control Flow

Control flow statements determine which code executes based on conditions. Strata supports if/else, while loops, and for loops.

If/Else Statements

let age: int = 25 if (age >= 18) { io.print("Adult") } else { io.print("Minor") }

Else If Chains

if (score >= 90) { io.print("A") } else if (score >= 80) { io.print("B") } else if (score >= 70) { io.print("C") } else { io.print("F") }

While Loops

Repeat code while a condition is true:

var i: int = 0 while (i < 5) { io.print(i) var i: int = i + 1 }

For Loops

C-style for loops with initialization, condition, and update:

for (var i: int = 0; i < 5; var i: int = i + 1) { io.print(i) }

Break and Continue

var i: int = 0 while (i < 10) { if (i == 3) { continue // Skip this iteration } if (i == 7) { break // Exit loop } io.print(i) var i: int = i + 1 }
