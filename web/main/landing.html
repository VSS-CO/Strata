Strata

A type-safe scripting language that combines the best of C, Go, Rust, and TypeScript. Compiles to C code or runs directly.

Documentation Try Playground

Key Features

Type-Safe

Compile-time type checking catches errors before execution. Explicit type annotations make code intentions clear and verifiable.

Immutable by Default

Variables are immutable (let/const) by default. Use var only when mutation is needed. Prevents entire classes of bugs.

Module System

Explicit, file-based module system with clear imports. Standard library under std:: namespace prevents shadowing.

Compiles to C

Generate efficient C code that you can inspect, modify, and compile with standard C toolchains. Zero runtime overhead.

Clean Syntax

Inspired by Go and Rust, Strata syntax is clean and expressive. No magic, no implicit conversions, explicit is better.

Single File

The entire compiler/interpreter fits in one file with zero external dependencies. Easy to understand, modify, and deploy.

Write Better Code

import io from std::io import math from std::math func factorial(n: int) => int { var result: int = 1 for (var i: int = 1; i <= n; var i: int = i + 1) { var result: int = result * i } return result } io.print(factorial(5))

Type-Safe Functions

  • Explicit Types: All parameters and returns have types
  • Compile-Time Checking: Errors caught before runtime
  • Module System: Clear imports with namespaces
  • Pure Functions: No hidden side effects
  • Immutable Default: var only when needed
