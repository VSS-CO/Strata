Syntax

Strata Extended uses a clean, modern syntax with explicit types and immutability by default. Programs execute from top to bottom.

Hello World

import io from str

io.print("Hello, World!")

Variables

Declare variables with explicit type annotations:

let x: int = 42
let name: string = "Strata"
var counter: int = 0  // mutable
const PI: float = 3.14159

Functions

Define functions with type annotations:

func add(a: int, b: int) => int {
  return a + b
}

func greet(name: string) => string {
  return "Hello, " + name
}

Control Flow

if/else, while, and for loops:

if (x > 10) {
  io.print("Large")
} else {
  io.print("Small")
}

while (counter < 5) {
  var counter: int = counter + 1
}

for (var i: int = 0; i < 10; var i: int = i + 1) {
  io.print(i)
}

Operators

Arithmetic, comparison, and logical operators:

a + b   // Addition
a - b   // Subtraction
a * b   // Multiplication
a / b   // Division
a % b   // Modulo

a == b  // Equality
a != b  // Inequality
a < b   // Less than
a > b   // Greater than

true && false  // Logical AND
true || false  // Logical OR
!true          // Logical NOT

Primitive Types

  • int – Integer numbers (42, -5)
  • float – Floating-point numbers (3.14, 2.0)
  • bool – Boolean values (true, false)
  • char – Single character ('A', 'x')
  • string – Text ("hello", 'world')
  • any – Any type (flexible, use with caution)
