Variables

Variables in Strata Extended are explicitly typed with let, const, or var. Type checking happens at compile-time before execution.

Immutable Bindings (let)

Use let for immutable variables (default):

let x: int = 42
let name: string = "Strata"
let pi: float = 3.14159

// x = 100  // ERROR: Cannot reassign let

Constants (const)

Use const for compile-time constants:

const MAX_SIZE: int = 1000
const PI: float = 3.14159

// const MAX_SIZE: int = 500  // ERROR: Cannot reassign const

Mutable Variables (var)

Use var for mutable variables that can be reassigned:

var counter: int = 0
var active: bool = true

var counter: int = 1  // OK: can reassign var
var counter: int = 2  // OK: can reassign var

Type Annotations

All variables require explicit type annotations:

let x: int = 42
let y: float = 3.14
let active: bool = true
let letter: char = 'A'
let message: string = "Hello"

Type Compatibility

  • int can be assigned to float
  • Any type can be assigned to any
  • String cannot be assigned to number (strict type checking)

Scoping

Variables have block scope:

let x: int = 10

if (true) {
  let y: int = 20
  io.print(x)  // OK: x is in scope
  io.print(y)  // OK: y is in scope
}

io.print(x)  // OK: x is still in scope
// io.print(y)  // ERROR: y is out of scope
