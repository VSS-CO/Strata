Modules

Strata Extended provides a standard library of built-in modules for I/O, math, and more. Use the import statement to access them.

Importing Modules

Import a module from the standard library with the from str syntax:

import io from str
import math from str

Using I/O (str.io)

Print and read values:

import io from str

io.print("Hello, World!")
io.print(42)
io.print(true)

Using Math (str.math)

Basic mathematical operations:

import math from str

let root: float = math.sqrt(16)   // 4.0
let abs: int = math.abs(-42)       // 42
let power: int = math.pow(2, 8)    // 256

Available Modules

  • str.io – Input/output
    • io.print(value: any) => void – Print a value
  • str.math – Mathematical functions
    • math.sqrt(n: float) => float – Square root
    • math.abs(n: int) => int – Absolute value
    • math.pow(base: int, exp: int) => int – Power

Complete Example

import io from str
import math from str

func main() => void {
  io.print("Enter a number:")
  
  let n: int = 16
  let root: float = math.sqrt(n)
  
  io.print("Square root of 16 is:")
  io.print(root)
}

main()
