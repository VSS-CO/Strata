Get Started

1. Install Strata

npm install
npx tsc

Compile TypeScript compiler to JavaScript.

2. Add Packages

strata add http-client
strata add crypto 2.1.0

Browse and install from the package registry.

3. Write Code

import client from http-client

let response = client.get("https://example.com")
io.print(response)

Use packages in your Strata programs.

4. Compile & Run

strata build
node dist/main.js

Compile to C, JavaScript, or execute directly.
